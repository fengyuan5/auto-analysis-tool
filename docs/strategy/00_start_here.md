# 从这里开始看

当前文档按下面顺序阅读：

1. [01_goals_and_closures.md](01_goals_and_closures.md)
   先看这份。
   作用：讲清楚这件事要做什么、对外输出什么、对内输出什么、闭环怎么转。

2. [02_execution_plan.md](02_execution_plan.md)
   第二份看这份。
   作用：讲清楚下一步怎么推进，先做什么，先证明什么。

3. [03_recipe_inventory.md](03_recipe_inventory.md)
   第三份看这份。
   作用：讲清楚 recipe 盘点结论、第一批试点选什么、每个样例接入时缺什么。

4. [04_v1_integration_spec.md](04_v1_integration_spec.md)
   最后看这份。
   作用：讲清楚第一版 recipe 接入规范和数据底座最小字段集。

5. [05_recipe_admission_rules.md](05_recipe_admission_rules.md)
   需要制定合入标准时看这份。
   作用：讲清楚以后内部和外部 recipe 样例合入时，必须满足哪些规则，才能支撑统一信息抽取和统一数据库。

配套表格：

1. [03_recipe_inventory.csv](03_recipe_inventory.csv)
   用来实际持续补样例盘点。

如果只想快速理解全局，先看第 1 份和第 3 份。
如果要开始落地做原型，再看第 2 份、第 4 份和第 5 份。
